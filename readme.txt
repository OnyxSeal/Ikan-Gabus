<h1>📚 FasilkomLib</h1>

<p><strong>FasilkomLib</strong> adalah aplikasi manajemen buku digital yang dibuat untuk mempermudah pengelolaan dan pencarian buku di lingkungan Fakultas Ilmu Komputer.</p>

<hr />

<h2>🔍 Fitur Utama</h2>
<ul>
  <li>🔐 Login &amp; Register Admin</li>
  <li>📖 Tambah, lihat, edit, dan hapus data buku</li>
  <li>🧾 Riwayat penambahan buku &amp; admin pencatat</li>
  <li>📂 Kategori buku dan pencarian</li>
  <li>📊 Dashboard sederhana untuk monitoring</li>
</ul>

<hr />

<h2>🚀 Teknologi yang Digunakan</h2>
<ul>
  <li>PHP Native</li>
  <li>MySQL</li>
  <li>HTML + CSS</li>
  <li>JavaScript</li>
</ul>

<hr />

<h2>🛠️ Cara Menjalankan</h2>
<ol>
  <li>Clone repositori:<br />
    <pre><code>git clone https://github.com/OnyxSeal/Ikan-Gabus.git</code></pre>
  </li>
  <li>Import database dari folder <code>database/fasilkomlib.sql</code> ke phpMyAdmin</li>
  <li>Sesuaikan konfigurasi koneksi di <code>connection/conn.php</code></li>
  <li>Jalankan melalui browser menggunakan XAMPP atau server lokal lainnya</li>
</ol>
